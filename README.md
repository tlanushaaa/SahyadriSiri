# 🌿 Sahyadri Siri – Sacred Water Monitoring Application

## 📌 Introduction

Sahyadri Siri is an Android-based environmental monitoring application developed for reporting and visualizing the condition of rivers, streams, waterfalls, and water bodies across the Western Ghats region.

The application provides a digital platform where users can submit water condition reports such as Clean, Polluted, or Dirty along with location details. The submitted reports are dynamically displayed on Google Maps using map markers for better environmental monitoring and visualization.

This project was developed as part of the internship titled:

**Android App Development using Generative AI**

---

# 🚨 Problem Statement

Water pollution and environmental degradation are increasing rapidly across many water bodies in the Western Ghats region. Traditional monitoring systems are often manual, slow, and lack digital accessibility.

There is a need for a simple Android-based application that enables users to:
- report water conditions
- visualize polluted locations
- encourage environmental awareness
- support community participation

- Google Maps Screen

- Geocoder Integration
